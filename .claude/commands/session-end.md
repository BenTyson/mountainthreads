# Session End

Before ending this session, perform the following:

1. Update `docs/session-start/CURRENT_STATUS.md` with:
   - What was completed this session
   - What's currently in progress
   - What should be done next
   - Any blockers or notes

2. Update `docs/PROJECT_PLAN.md` checkboxes for any completed tasks

3. Create or update the session log in `docs/sessions/SESSION_XXX.md` with:
   - Date
   - Goals for this session
   - What was completed
   - Files created/modified
   - Next steps
   - Any important notes or decisions

4. Provide a brief summary of what was accomplished and what the next session should focus on.
